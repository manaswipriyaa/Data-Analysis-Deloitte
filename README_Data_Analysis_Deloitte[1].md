# Data Analysis — Deloitte Job Simulation

A virtual job simulation completed on Forage, replicating a real-world data analytics engagement at Deloitte. The project involved analysing business data, building dashboards, and presenting data-driven recommendations.

---

## About the Simulation

**Company:** Deloitte
**Platform:** Forage
**Completed:** February 2026
**Focus Area:** Data Analytics, Business Intelligence

This simulation placed me in the role of a data analyst at Deloitte, working through tasks that mirror actual client engagements — from data interpretation to visual storytelling and strategic recommendations.

---

## Tasks Completed

### Task 1 — Data Analysis
- Analysed a provided business dataset to identify trends, anomalies, and key insights
- Used Excel and Python to explore and summarise the data
- Identified the most significant drivers affecting the client's business metrics

### Task 2 — Dashboard Creation
- Built a visual dashboard to communicate findings clearly to a non-technical audience
- Focused on clarity, storytelling, and actionable insight presentation

### Task 3 — Business Recommendations
- Synthesised analysis into a structured set of recommendations
- Presented findings in a format suitable for a client-facing report
